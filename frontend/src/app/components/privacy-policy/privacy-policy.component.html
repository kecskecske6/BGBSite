Privacy Policy

Usage of data

The bot may use stored data, as defined below, for most of its features.
Data is not shared with any 3rd-party site or service.

Stored information

Servers

The bot may store the following information about a server:
  • serverId: The server's ID
  • commandChannels: IDs of channels of the server where users are able to use the bot's commands
  • roles: IDs of roles that have been set to be automatically given to users by the bot on certain events
  • dropBlacklist: IDs of channels of the server where the bot's drop feature is disabled
  • xpBlacklist: IDs of channels of the server where the bot's ranking system is disabled

Users

The bot may store the following information about a user:
  • userId: The user's ID
No other Discord-related information outside of the above mentioned will be stored.

Updating data

The data may be updated when using specific commands or features of the bot.
Updating data will require the input of an end user, and data that can be seen as sensitive, such as content of a message, may need to be stored when using certain commands.

Removal of data

Manual removal of the data can be requested through the bot's official support server.
For security reasons the owner of the bot will ask you to provide us with proof of ownership of the server, that you wish the data to be removed of. Only a server owner may request manual removal of data and requesting it will result in the bot being removed from the server, if still present on it.
